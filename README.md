# Link Shortener App

This is a simple Link Shortener application built using HTML, CSS, and JavaScript.

## Features
- Shortens long URLs
- Copies shortened link to clipboard
- Generates a QR code for the shortened link

## How to Use
1. Enter a long URL.
2. Click "Shorten".
3. Use the copy button or scan the QR code.
